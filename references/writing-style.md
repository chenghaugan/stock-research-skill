# 写作风格规范

> 以资深股票投资经理的视角撰写 — 专业、克制、有判断力。

---

## 一、禁止的 AI 句式

以下表达**绝对禁止**出现在报告中，一旦出现视为质量不合格：

| 禁止句式 | 问题 | 正确做法 |
|---------|------|---------|
| "值得注意的是..." | 过度强调，含糊其辞 | 直接陈述事实 |
| "首先、其次、最后"滥用 | 机械罗列 | 按逻辑主线叙述 |
| "我们可以看到..." | 强行代入读者 | "本文认为"、"我们判断"或直接陈述 |
| "综上所述..." | 套话结尾 | 结论性语句，无需过渡词 |
| "本文将从以下几个维度..." | 模板套话 | 直接进入分析 |
| "随着...的不断发展..." | 正确的废话 | 用具体数据和趋势说话 |
| "既有优势也有劣势" | 和事佬 | 明确优势是什么，劣势是什么，哪方占主导 |

## 二、写作原则

### 2.1 有明确观点
- YES: "公司核心优势在于渠道壁垒，但新业务拓展能力不足将在未来2-3年压制估值"
- NO: "公司既有优势也有劣势，需要辩证看待"
- 避免主观表述如"我认为"、"我觉得"，用客观叙述代替：用"我们判断"、"从产业逻辑看"、"数据显示"等替代

### 2.2 用数据和逻辑说话
- 所有判断必须有财报数据、行业数据或逻辑推演支撑
- 禁止"预计未来有望增长"类无锚点表述
- 给出具体预测区间和关键假设

### 2.3 有立场有偏好
- 明确表述看好/不看好理由及程度
- 估值结论必须给出具体锚点和参考系

### 2.4 克制简洁
- 少用形容词、副词，多用动词和名词
- 一句话说清的事，绝不拆成三句
- 避免"一定程度上"、"相对而言"等模糊修饰

---

## 三、深度强制规则（核心）

> **这是报告质量的生命线。每一条关键判断必须满足以下规则，缺一则该段落不合格。**

### 3.1 三段论推导（硬性要求）

每条关键判断必须走完三层，才能写入报告：

```
数据陈述（是什么）
  → 因果分析（为什么会这样）
    → 投资含义（对投资意味着什么）
```

**自检方式**：读完任何段落，如果读者只能在脑中回答"哦，知道了"，而不是"所以XX"，则该段落不合格。

**示例对比：**

| 层级 | ❌ 不合格（停留第一层） | ✅ 合格（走完三层） |
|------|---------------------|-------------------|
| 数据陈述 | "公司IDC收入同比增长41.8%。" | 同左 |
| 因果分析 | （缺失） | "主要受益于廊坊固安项目首批机柜交付上架，该订单来自字节跳动2024年大规模招标。" |
| 投资含义 | （缺失） | "这个增速在同行中处于领先水平，意味着公司在AI智算中心布局上已率先兑现为收入。但需注意，单一大客户集中度高意味着续约风险也相应放大。" |

**三层缺一不可**。只给数据不给含义 = 信息堆砌；只给含义不给数据 = 空中楼阁。

### 3.2 因果链完整呈现

从数据到结论，必须展示完整的推理链条，禁止简单"因为所以"：

- ❌ "公司负债率74.7%，因此财务风险高"
- ✅ "公司负债率74.7%，其中有息负债率44.22%，流动比率仅0.35。这意味着：一是每年利息支出侵蚀约X亿净利润；二是若融资渠道受阻，在建项目可能面临停工风险。这是我们将其列为核心风险的根本原因"

### 3.3 公司分析顺序

分析任何公司时，遵循以下顺序：
1. **先说定位和地位**：公司在产业链中处于什么位置，行业排名/市场份额
2. **再说数据支撑**：用财务数据、业务数据印证定位判断
3. **最后给出判断和理由**：基于前两步得出明确结论，不模棱两可
4. **说明投资逻辑的完整推导过程**：从行业判断到公司选择到估值锚定的全链条

## 四、段落比例硬约束

### 4.1 段落 vs 表格比例

- **每个章节段落文字占比 ≥ 80%**
- **表格仅限以下两类场景**：
  - 竞对公司财务横向对比（竞争力分析章节）
  - 风险分级表（风险关注章节）
- 其他所有内容（行业分析、业务拆解、投资逻辑、估值讨论等）必须用段落叙述

### 4.2 段落长度要求

- 单段不少于3句。禁止"单句成段"——一句话写个标题下面全是表的做法。
- 长段落（超过8句）应在段内用句号自然分句，保持可读性，但不得拆成碎片。
- 段落之间用逻辑连接词或过渡句串联。

### 4.3 数据呈现方式

数据必须**嵌入叙述之中**，后紧跟分析含义，不得孤立列出：

- ❌ "毛利率：32%。净利率：15%。ROE：18%。"
- ✅ "毛利率32%，较去年同期提升2个百分点，主要驱动来自产品结构向高端化倾斜。净利率15%处于行业中上水平，ROE 18%则反映了公司较高的资产周转效率——这不是一家靠高杠杆撑ROE的公司。"

---

## 五、连贯性强制要求

### 5.1 章节过渡

- **章末必须有1句过渡**，连接到下一章。如"理解了行业格局，接下来看公司在其中的位置。"
- **章首必须有1句承接**，说明"为什么接下来要讲这个"。如"行业空间毋庸置疑，但能否转化为公司业绩，取决于竞争力。"
- 禁止"突然换话题"——读者拆开任意两章相邻段落，应能看出逻辑关联。

### 5.2 全文主线

全文贯穿一条逻辑主线，读者在报告任意位置都可知其位置：

```
行业逻辑（钱在哪，怎么赚）
  → 公司选择（为什么是这家，凭什么赢）
    → 估值锚定（市场在为什么定价，错在哪）
```

每章开头用一句话锚定其在主线中的位置。

### 5.3 自然过渡技法

- 使用自然过渡语串联逻辑："回过头看"、"把这些数据放在一起看"、"沿着这个逻辑往下走"
- 适当使用设问引导思考："这意味着什么？""为什么市场对此视而不见？"
- 避免机械的章节编号式过渡（"以上是行业分析，以下是竞争力分析"）

## 六、逆向思辨要求

> **投资报告的核心价值不是罗列事实，而是提供与众不同的判断。** 如果报告结论与市场共识一致，报告没有存在价值。

### 6.1 必须回应的问题

投资逻辑章节必须包含：

1. **"市场为什么错"** — 先解释当前估值的合理性（市场不傻，定价有其逻辑），然后说明错在哪（我们凭什么不一样）
2. **"什么情况下我们的判断是错的"** — 设定可验证的证伪条件，不自说自话
3. **至少回应一个主流分歧观点** — "市场上有人担心XX，我们的看法是..."

### 6.2 示例

- ❌ "我们认为公司被低估。"（无对立面，等于没说）
- ✅ "市场给予公司15x P/E，核心担忧是大客户依赖度过高。这个担心有其合理性——前三大客户占比达65%。但市场忽略了两个事实：一是公司与头部客户的合同已锁价至FY27，短期无流失风险；二是公司主动降低单一客户占比的策略已见成效，FY25新客户收入占比从5%提升至18%。我们认为市场过度定价了集中度风险，而低估了新客户拓展的加速度。"

## 七、重要信息强调规范

| 内容类型 | 强调方式 |
|---------|---------|
| 核心判断/结论 | **加粗** |
| 关键数据 | **加粗** |
| 致命风险 | <font color="red">**加粗+红色**</font> |
| 投资评级 | **加粗** |
| 关键假设 | **加粗** |
| 管理层关键表述 | **加粗引用** |

**强调原则**：
- 全文加粗率 5-10%，不超过 10%
- 致命风险必须红色标注
- 强调必须有信息价值，不为强调而强调

### 7.1 ⚠️ 关键：HTML 标签内的 `<` 必须转义

> **这是 Obsidian 渲染的硬性约束，违反将导致报告大面积渲染失败。**

当 `<font color="red">` 或任何 HTML 标签**内部**出现"小于"含义的 `<` 字符时（如 `营收增速<15%`、`流动比率<0.5`、`PE<20x`），必须写成 `&lt;`，否则 Obsidian 会将 `<` 误解析为新 HTML 标签的开始，导致从该位置起**所有后续 Markdown 内容渲染失效**。

| 场景 | ❌ 错误写法 | ✅ 正确写法 |
|------|-----------|-----------|
| 小于号 | `流动比率<0.5` | `流动比率&lt;0.5` |
| 小于号 | `营收增速<15%` | `营收增速&lt;15%` |
| 小于号 | `净利率<8%` | `净利率&lt;8%` |
| 小于号 | `PE<20x` | `PE&lt;20x` |
| 纯数值引用块（无 `<font>`） | `营收增速<15%` 正常 | 无需转义 |

> 💡 **判断方法**：如果 `<` 出现在 `<font ...>` 和 `</font>` 之间，就必须转义。如果 `<` 在普通 Markdown 文本中（不在 HTML 标签内），则不需要转义。

## 八、段落写作范例

**❌ AI风格（禁止）**：
> 综上所述，我们认为该公司具有良好的发展前景。从行业角度来看，公司所处行业市场规模较大，增长速度较快。同时，公司具有较强的核心竞争力，在技术、渠道、品牌等方面具有明显优势。我们首次覆盖，给予"推荐"评级。

**✅ 投资经理风格（要求）**：
> 公司是XX环节的核心供应商，过去三年该业务收入CAGR达35%，高于行业平均15个百分点。管理层在FY24业绩交流会中明确表示将资源向XX业务倾斜，我们判断这一战略选择正确。当前P/E 18x处于历史估值下沿，较可比公司平均25x有明显折价——市场在担心什么，我们后续分析。

## 九、风险分析写作范例

**❌ AI风格（禁止）**：
> 公司存在一定的投资风险。主要风险包括行业竞争加剧、政策不确定性、以及管理层变动等。投资者需要密切关注这些风险。

**✅ 投资经理风格（要求）**：
> **风险分级：**
> - <font color="red">**致命风险：政策转向可能导致新业务审批周期延长50%以上，投资逻辑失效。监控阈值：政策细则发布后审批周期&gt;6个月则触发退出。**</font>
> - **重大风险：头部竞品加大投入，市场份额可能下滑5个百分点。触发后需重新评估估值，目标价下调10-15%。**
> - **一般风险：核心技术人员流失风险，需关注离职率变化。**

## 十、估值讨论写作范例

**❌ AI风格（禁止）**：
> 综合以上分析，我们认为公司具有较好的投资价值。估值方面，采用P/E估值法，给予XX倍市盈率，对应目标价XX元。

**✅ 投资经理风格（要求）**：
> **估值讨论：**
> 
> 公司当前P/E 18x，处于历史估值区间的下沿（历史区间15-30x，中位22x）。同行可比公司平均25x，溢价来自市场给予龙头更高的确定性溢价。
> 
> **乐观情景下**（概率约20%）：如果新业务FY26收入占比突破25%+行业增速维持15%以上，市场可能从"怀疑"切换到"认可"，估值有望修复至22-25x区间，对应股价上行空间约30-40%。驱动逻辑：新业务占比提升将改变市场对公司的定位——从"传统XX商"重估为"XX+新业务"双轮驱动。
> 
> **悲观情景下**（概率约20%）：如果行业增速放缓至个位数+新业务推进不及预期（收入占比<10%），估值可能进一步压缩至12-15x。此时公司的"传统业务天花板+新业务未兑现"双重压力叠加，市场可能以"价值陷阱"定价。
> 
> **中性情景下**（概率约60%）：新业务稳步推进至15-20%占比，行业增速维持10-12%，估值在18-22x区间波动，对应股价在当前水平上下10%范围内震荡。
> 
> **核心分歧**：市场给予低估值，本质是对"新业务能否兑现"投了不信任票。我们的判断与市场不同之处在于：一是XX合同已锁定FY26收入的40%，确定性高于市场预期；二是管理层过往3次转型均兑现，执行力的历史记录不应被忽视。
>
> **安全边际**：即便悲观情景实现，当前价位下跌空间控制在30%以内；乐观情景上行40%。风险收益比约为1:1.3，在当前的价位下属于可接受范畴。
